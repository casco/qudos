De forma similar a la compuerta <a href={`${import.meta.env.BASE_URL}/gates/x-axis`}>Pauli-X</a> implica una rotación en el eje $x$ de la esfera de Bloch, esta compuerta se acompaña con la rotación empleada. Imitar el comportamiento de la compuerta Pauli-X implica aplicar una RotX con $\pi$ radianes de rotación.

Esta compuerta permite abstraerse de ángulos fijos, permitiendo una mayor libertad de aplicación.

En forma de matriz se puede observar de la siguiente manera:
```math
RotX(\theta)=
\begin{bmatrix}
cos\left(\frac{\theta}{2}\right)&-i sin\left(\frac{\theta}{2}\right)\\
-i sin\left(\frac{\theta}{2}\right)&cos\left(\frac{\theta}{2}\right)
\end{bmatrix}
```

En qiskit se puede implementar con la funcion `rx`.

<Code
  code={`
from qiskit import QuantumCircuit
import numpy as np
pi = np.pi
qc = QuantumCircuit(1)
qc.rx(pi, 0) # aplica una rotacion de pi al qubit 0 en el eje x
`

}
