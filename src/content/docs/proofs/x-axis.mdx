## Conversiones de estados comunes

Recordando los valores de los Qubits en sus estados comunes (o bases), a continuacion se detallaran las pruebas matematicas de las conversiones de un estado comun a otro utilizando la <a href={`${import.meta.env.BASE_URL}/gates/x-axis`}>puerta NOT cuantica</a>.

#### $\ket{0} \overset{X}{\rightarrow} \ket{1}$

```math
\begin{align}
X\ket{0}&=\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
1\\
0
\end{bmatrix}\\
&=\begin{bmatrix}
0\\
1
\end{bmatrix}=\ket{1}
\end{align}
```

#### $\ket{1} \overset{X}{\rightarrow} \ket{0}$

```math
\begin{align}
X\ket{1}&=\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
0\\
1
\end{bmatrix}\\
&=\begin{bmatrix}
1\\
0
\end{bmatrix}=\ket{0}
\end{align}
```

#### $\ket{+} \overset{X}{\rightarrow} \ket{+}$

```math
\begin{align}
X\ket{+}&=\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
1\\
0
\end{bmatrix}+\begin{bmatrix}
0\\
1
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
1\\
0
\end{bmatrix}+\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
0\\
1
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
0\\
1
\end{bmatrix}+
\begin{bmatrix}
1\\
0
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}(\ket{1}+\ket{0})\\
&=
\frac{1}{\sqrt{2}}(\ket{0}+\ket{1})=\ket{+}
\end{align}
```

#### $\ket{-} \overset{X}{\rightarrow} \ket{-}$

```math
\begin{align}
X\ket{-}&=\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
1\\
0
\end{bmatrix}-\begin{bmatrix}
0\\
1
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
1\\
0
\end{bmatrix}-\begin{bmatrix}
0&1\\
1&0
\end{bmatrix}\begin{bmatrix}
0\\
1
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}\left(\begin{bmatrix}
0\\
1
\end{bmatrix}-
\begin{bmatrix}
1\\
0
\end{bmatrix}\right)\\
&=
\frac{1}{\sqrt{2}}(\ket{1}-\ket{0})\\
&=
\frac{1}{\sqrt{2}}(-\ket{0}+\ket{1})\\
&=
\frac{1}{\sqrt{2}}(\ket{0}-\ket{1})=\ket{-}
\end{align}
```

El paso del 14 al 15 se cumple por quitar como factor el negativo y obviando la fase global.
